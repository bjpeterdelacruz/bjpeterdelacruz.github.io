utils.linked_list.push(head, data) → utils.linked_list.Node[source]

This function creates a new node if head is None. If data is not None, a new node is created, and the next pointer is set to head, thus making the new node the new head of the linked list.

Parameters

  • head – The head of a linked list

  • data – The data to add to the linked list

Returns

A new node that will be the new head of the linked list
