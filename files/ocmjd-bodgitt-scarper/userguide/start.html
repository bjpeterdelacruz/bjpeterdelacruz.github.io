Modes

You can run the application in one of three modes: standalone client, networked client, or server. Type any one of the following commands on the command line to start up the application in the appropriate mode.

Standalone Client: java -jar runme.jar alone
Networked Client: java -jar runme.jar
Server: java -jar runme.jar server

Files

The following files will be created in the same directory in which the application is started:

client.log The log file that standalone and networked clients use. Located in the logs subdirectory.
server.log The log file that servers use. Located in the logs subdirectory.
client_shutdown.log The log file that standalone and networked clients use when the application is shutting down. Located in the logs subdirectory.
server_shutdown.log The log file that servers use when the application is shutting down. Located in the logs subdirectory.
suncertify.properties The file to which the following information will be saved:

main.window.height The height of the main application window
main.window.width The width of the main application window
networked.server.port The port number on the server that a networked client uses to connect to it
networked.server.location The server location
server.autostart If true, the server will be started automatically when running in server mode. If false, you will have to start the server manually.
server.db.location The location of the database file on the server
server.port The port number to which networked clients will use to connect to the server
standalone.db.location The location of the database file on your computer

Important Notes

  • The application can run on any operating system (Windows, Mac OS X, or Linux) as long as Java SE 6 or higher is installed. The screenshots in this User Guide show the application running on OS X Mountain Lion.
